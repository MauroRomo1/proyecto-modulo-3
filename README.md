<h3>Para clonar este proyecto copie y pege uno por uno los siguientes comandos en una terminal</h3>

- Paso 1:
```
git clone https://github.com/MauroRomo1/proyecto-modulo-3.git
```

- Paso 2:
```
npm install
```
- Paso 3:
```
npm run dev
```
